函数

函数的定义与基本概念

函数是组织好的、可重复使用的、用来实现单一或相关联功能的代码段。Python 中的函数使用 def 关键字定义。

参数类型详解

  • 位置参数：根据参数位置进行传递
  • 默认参数：在定义函数时给参数指定默认值
  • 可变位置参数：使用 *args 接收任意数量的位置参数
  • 可变关键字参数：使用 **kwargs 接收任意数量的关键字参数

变量作用域

Python 中有四种作用域：

  • 局部作用域：在函数内部定义的变量
  • 嵌套作用域：在嵌套函数中定义的变量
  • 全局作用域：在模块顶层定义的变量
  • 内置作用域：Python 内置的变量

lambda

Lambda 函数是一种小的匿名函数，可以接受任意数量的参数，但只能有一个表达式。

闭包

闭包是由函数及其相关的引用环境组合而成的实体

装饰器

装饰器是一种修改函数或类行为的函数，它接受一个函数作为参数，并返回一个新的函数

面向对象编程

类与对象的基本概念

类是对现实世界中事物的抽象，对象是类的实例

继承

继承允许我们定义一个类，它继承另一个类的所有属性和方法

封装

封装是将数据和对数据的操作包装在一起，并隐藏内部实现细节

多态

多态允许不同类的对象对同一消息做出不同的响应

特殊方法

Python 中的特殊方法以双下划线开头和结尾，用于实现类的特定行为

模块

模块的创建与导入

模块是一个包含 Python 代码的文件，以 .py 为扩展名

包

包是一种组织模块的方式，是一个包含多个模块的目录，且必须包含一个 __init__.py 文件

迭代器/装饰器/生成器

迭代器

迭代器是实现了迭代器协议的对象，包含 __iter__() 和 __next__() 方法

生成器

生成器是一种特殊的迭代器，使用 yield 语句返回数据

装饰器

装饰器是一种修改函数或类行为的函数

其他常用库简介

  • NumPy：科学计算库，提供多维数组对象和数学函数
  • Pandas：数据分析库，提供数据结构和数据分析工具
  • Matplotlib：绘图库，创建静态、动态和交互式可视化
  • Flask/Django：Web 框架，用于构建 Web 应用程序
  • BeautifulSoup：HTML/XML 解析库，用于网页抓取
  • Pillow：图像处理库，提供图像处理功能
  • SQLAlchemy：SQL 工具包和对象关系映射（ORM）工具
