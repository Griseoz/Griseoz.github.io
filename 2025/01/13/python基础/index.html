什么是 python

Python 是一门高级程序设计语言，也属于解释型的语言。

python 的特点

  • 语法简洁
  • 类库强大
  • 胶水语言（调用其他语言的类库）
  • 代码量较少

python 语法

在 python 编程时要注意缩进

变量

与 C，Java 等其他语言不同的是 Python 没有用于声明变量的命令。所以在 Python 中，当你给它赋值时就会创建变量。如果要指定变量的数据类型，可以通过强制转换来完成。

变量命名规则

  • 变量名必须以字母或下划线字符开头
  • 变量名不能以数字开头
  • 变量名称只能包含字母数字字符和下划线（Az、0-9 和 _ ）
  • 变量名区分大小写（age、Age 和 AGE 是三个不同的变量）

数据类型

  • 整数：可以为任意大小、包含负
  • 浮点数：就是小
  • 字符串：以单引号 ‘、双引号”、三引号 ‘’’ 或 “””括起来的文
  • 布尔：只有 True、False 两种
  • 空值：用 None 表
  • 变量：是可变
  • 常量：不可变

运算符

格式化字符串

format

  • format 内的参数起始编号以 0 开始，即花括号引用的 0 表示第一个参数，1 表示第二个参数。
  • format 还可以根据关键词引用，这种方式就不需要考虑参数在 format 内的哪个位置，只根据关键词引用
  • format 方法调到用字符串可以直接调用数字，不需要额外进行数字转字符串的操作

f’ 字符串

  • f’ 字符串引用已定义的字符串变量
  • 还可以使用 冒号+点+数字+f 来指定浮点数在格式化是保留几位小数

列表

特点：

  • 可变：创建后可以修改、添加或删除元素
  • 有序：元素有明确的顺序，可以通过索引访问
  • 可重复：元素可以重复
  • 可包含任意类型：一个列表可以包含数字、字符串、列表、字典等不同类型
