ES6

ES 全称 ECMAScript ，是 JS 的语言标准。而 ES6 是新的 JS 语法标准。

变量声明

const、let、var 区别

  • var：，ES5 和 ES6 中，定义全局变量（是 variable 的简写）。
  • let：定义局部变量，替代 var。
  • const：定义常量（定义后，不可修改）。
  1. var

    用 var 定义的全部变量，有时候会污染整个 js 的作用域

  1. let

    用 let 声明减少 var 声明带来的污染全局空间

用 let 声明变量：（不能打印结果）

  1. const

    变量是希望声明后，在业务层就不再发生变化，此时可以用 const 来定义

变量的解构赋值

通过数组或者对象的方式，对一组变量进行赋值，这被称为解构。解构赋值在实际开发中可以大量减少我们的代码量，并且让程序结构更清晰

  1. 数组的解构赋值

在解构赋值时，是允许使用默认值的
