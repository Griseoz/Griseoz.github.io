jQuery

jQuery 就是 JavaScript 的一个库，里面封装了很多预定义的函数，比如获取元素，执行隐藏、移动等，目的就 是在使用时直接调用，不需要再重复定义，这样就可以极大地简化了 JavaScript 编程

常用方法

  1. $.each()

    个通用的迭代函数，它可以用来无缝迭代对象和数组。数组和类似数组的对象通过一个长度属性（如一个函数的参数对象）来迭代数字索引，从 0 到 length - 1，其他对象通过其属性名进行迭代。

  1. $.trim()

    去掉字符串起始和结尾的空格。

  1. $.typr()

    确定 JavaScript 对象的类型。

  1. $.isArray()

    用来测试指定对象是否为一个数组

  1. $.isFunction()

    用来测试指定对象是否为一个函数

属性

  1. attr()

    专门操作属性值为非布尔值的属性，该方法读写一体

  1. prop()

    专门操作属性值为布尔值的属性，该方法读写一体。

  1. val()

    获取表单元素的值和设置表单元素的值，该方法读写一体
